Materias Extras


Desenvolvi de software
Cursadas
Metodologias Ágeis
Estudo das metodologias ágeis, como Scrum e Kanban, e como aplicá-las no desenvolvimento de software para melhorar a flexibilidade, comunicação e produtividade
Gerenciamento de Projetos de Software
Técnicas para planejar, monitorar e controlar projetos de software, incluindo cronogramas, orçamentos, recursos e riscos.
Engenharia de Requisitos
Foco na coleta, análise, documentação e validação dos requisitos de software para garantir que o produto final atenda às necessidades dos usuários.
Testes de Software
Fundamentos dos testes de software, incluindo testes unitários, de integração, de sistema e de aceitação, para garantir a qualidade do produto.
Arquitetura de Software
Estudo da estrutura e design de sistemas de software complexos, incluindo padrões de arquitetura, como MVC, microserviços, e arquitetura em camadas.
Análise e Modelagem de Sistemas
Técnicas para representar e modelar sistemas de software, usando diagramas como UML (Unified Modeling Language) e diagramas de casos de uso.
Design de Software
Foco nas práticas e princípios de design de software, incluindo design orientado a objetos, SOLID, e design patterns (padrões de projeto).
Desenvolvimento Web
Técnicas de desenvolvimento de aplicações web, incluindo front-end (HTML, CSS, JavaScript) e back-end (servidores, APIs, bancos de dados).
Desenvolvimento de Software para Dispositivos Móveis
Práticas e ferramentas para o desenvolvimento de aplicativos móveis, como Android e iOS, incluindo desenvolvimento nativo e multiplataforma.
DevOps
Estudo da integração entre desenvolvimento e operações, visando a automação de processos, integração contínua (CI), entrega contínua (CD) e monitoramento.

Linguagens de Programação


Linguagens
Python
é uma linguagem de programação de alto nível, interpretada e de propósito geral. Ela é conhecida pela sua sintaxe simples e legibilidade, o que a torna ideal para iniciantes, mas também poderosa o suficiente para desenvolvedores experientes.
Java
Java é uma linguagem de programação orientada a objetos, robusta e multiplataforma. O famoso slogan “escreva uma vez, execute em qualquer lugar” refere-se à sua capacidade de ser executada em qualquer dispositivo que tenha a JVM (Java Virtual Machine).
Javascript
é uma linguagem de programação amplamente usada no desenvolvimento web para criar páginas dinâmicas. É uma linguagem baseada em eventos e orientada a objetos.
Ruby
é uma linguagem de programação de alto nível, dinâmica e orientada a objetos. Ela é projetada para ser simples e flexível.
PHP
PHP é uma linguagem de script server-side amplamente utilizada para desenvolvimento web. Originalmente projetada para gerar páginas web dinâmicas.
C
C é uma das linguagens de programação mais antigas e influentes, conhecida por sua eficiência e controle sobre o hardware.
C++
é uma linguagem de programação que estende o C, adicionando suporte para programação orientada a objetos, abstração e manipulação de memória direta.
C#
é uma linguagem de programação desenvolvida pela Microsoft, parte do .NET framework. É orientada a objetos e é popular para desenvolvimento de software para a plataforma Windows.
